# Paradise Nursery Shopping App

Paradise Nursery is a React and Redux shopping cart application for buying houseplants. The project includes a landing page, company information, categorized plant listings, and a fully functional cart page.

## Project Name Details

Project name: Paradise Nursery

The application is designed for a fictional online nursery that sells indoor plants grouped by category. Users can browse plants, add items to the cart, update quantities, remove items, continue shopping, and see the total cart cost.

## Main Features

- Landing page with the Paradise Nursery company name and a Get Started button.
- About Us page with details about the company.
- Product listing page with three plant categories and six unique plants per category.
- Add to Cart buttons that update the cart and become disabled after use.
- Shopping cart page with quantity controls, delete buttons, item totals, and total cart amount.
- Shared navigation bar for Home, Plants, and Cart with a dynamic cart count.

## Run Locally

```bash
npm install
npm run dev
```
